# Engine

Game Engine developed by Miquel Izquierdo & Guillermo Pinto, students from UPC-CITM.
This engine will allow you to create your own 3D games in a simpler way.

For the moment, this engine is in develpment...

Controls: 
-Move Camera: W,A,S,D

Libraries used:
-Open GL

-ImGui

-MathGeoLib

-JSON Parson
